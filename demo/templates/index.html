LifeSim generates coherent long-horizon life trajectories for users, modeling cognitive states (intent + motivation) at each node to capture behavioral variation across contexts.

Live Generation Preset Demo

Key Features

Long-horizon Trajectory Generation

Generates coherent life event sequences spanning multiple domains, with each node carrying full temporal, spatial, and weather context.

BDI Cognitive State Modeling

Each node carries explicit intents and implicit needs, driving differentiated interaction behavior via the Belief-Desire-Intention model.

Node-driven Interaction

Select any node to chat with the user in that cognitive state. Switch nodes to compare behavioral differences across contexts.

How It Works

1
Select a trajectory
Choose a preset sequence from the sidebar — the map instantly renders the full trajectory and user profile.
2
Browse the map
Follow the user's movement path; semi-transparent dashed lines connect each life node.
3
Click a node to inspect cognitive state
A detail drawer slides in showing the event description, explicit intents, and implicit needs.
4
Chat with the user at that moment
Click "Talk to User" to interact with the user in that cognitive state. Switch nodes to contrast behaviors.
